Prerequisites

  1. Basic understanding on Git and GitHub. Checkout tutorial here
  2. Basic understanding on Markdown language. Checkout tutorial here

Detailed instructions:

  • Create a GitHub account.
  • Send email to Prof. Mike Heroux for permission on the repo.
  • Go to https://github.com/Collegeville/Homework to clone the repo.
  • Check out the given exercises on the website.
  • Go to directory homework-submission and create your own folder with your full name is the folder's name.
  • Finish the home work, put in your folder, then push.

Plagiarism

All students are expected to be honest on doing their own homework. See the policy here and here . Plagiarism will be result in failing the course.
