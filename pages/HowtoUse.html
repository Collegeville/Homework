Prerequisites

  1. Basic understanding on Git and GitHub. Checkout tutorial here
  2. Basic understanding on HTML and Markdown language.

Detailed instructions for submission:

  • Create a GitHub account.
  • Send email to Prof. Mike Heroux for permission on the repo.
  • Go to https://github.com/Collegeville/Homework to clone the repo.
  • Check out the given exercises on the website.
  • Go to directory homeworkSubmission and create your own folder with your full name is the folder's name.
  • Finish the home work, put in your folder; other folders should not been touch
  • Commit and push to the GitHub repo. See detailed instructions below.
